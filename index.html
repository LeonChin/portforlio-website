about

Hey, my name is Leon. I live in Canberra, Australia and am currently building web applications using the React.js framework.My first bit of exposure to web development was from one of my master courses named ART AND INTERACTION IN NEW MEDIA. In there, I learned to use basic HTML, CSS and P5.js library to create web animations. I'd like to think that I started my web developer journey since then.

Check out some of my work, and feel free to shoot me an email if you'd like to work together.
